## משימת תכנות
Tic Tac Toe Game:

a. Development language: of your choice

b. Development paradigm: Preference for object-oriented but non-binding programming

c. Description and settings: There is an opening game X circle when there is a player playing in front of the computer.
The goal is to not be able to beat the computer
There is no need to develop a GUI, you can create a basic interface that 0 represents a circle and 1 represents an X
